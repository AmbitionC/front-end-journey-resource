下表总结了原始值和引用值的主要区别：

| 特性 | 原始值 | 引用值 |
| :---: | --- | --- |
| 存储位置 | 存储在栈内存中 | 存储在堆内存中 |
| 存储方式 | 直接存储实际的值 | 存储的是值的引用（地址） |
| 复制 | 值的复制是独立的 | 复制的是引用，指向同一块内存区域 |
| 比较 | 使用值本身进行比较 | 比较的是引用地址，判断是否指向同一块内存区域 |
| 可变性 | 不可变，修改值会创建一个新的值 | 可变，修改对象的属性不会改变引用地址，但会改变对象内部的状态 |
| 示例 | `let num = 10;` | `let obj = { name: 'Alice' };` |


#### 原始值
原始值存储在栈内存中，它们的值直接存储在变量访问的位置。因此，复制原始值时，实际上是复制了值本身，每个变量都有自己独立的存储空间。常见的原始值包括数字、字符串、布尔值、null 和 undefined。



#### 引用值
引用值存储在堆内存中，变量保存的是值的引用地址。复制引用值时，复制的是引用地址，多个变量可以指向同一块内存区域。因此，修改其中一个变量会影响其他变量。常见的引用值包括对象、数组、函数等。



#### 区别
综上所述，原始值和引用值在存储位置、存储方式、复制、比较、可变性等方面有着明显的区别。理解这些区别对于正确使用和处理 JavaScript 中的数据至关重要。
