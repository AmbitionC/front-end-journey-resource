下表总结了一些常用的 BOM（浏览器对象模型）属性、对象和方法：

| 类别 | 名称 | 描述 |
| :---: | --- | --- |
| **窗口属性** | `window.innerWidth` | 窗口的内部宽度（以像素为单位），不包括滚动条。 |
| | `window.innerHeight` | 窗口的内部高度（以像素为单位），不包括滚动条。 |
| | `window.location` | 提供了有关当前窗口的信息，并且可以用来导航到其他页面。 |
| **文档对象** | `document.getElementById()` | 根据元素的 ID 返回对应的元素对象。 |
| | `document.querySelector()` | 返回文档中与指定的 CSS 选择器或者一组 CSS 选择器匹配的第一个元素。 |
| | `document.createElement()` | 创建指定的 HTML 元素或者一个 HTMLUnknownElement。 |
| **导航对象** | `history.back()` | 在历史记录中后退一个页面。 |
| | `history.forward()` | 在历史记录中前进一个页面。 |
| **定时器方法** | `setTimeout()` | 设置在指定的毫秒数后执行函数或指定的代码一次。 |
| | `setInterval()` | 设置在指定的毫秒数间隔周期性地执行函数或指定的代码。 |
| **警告框方法** | `alert()` | 显示一个带有指定消息和一个确认按钮的警告框。 |
| | `confirm()` | 显示一个带有指定消息和一个确认按钮以及一个取消按钮的对话框。用户可以点击其中一个按钮作出选择。 |
| | `prompt()` | 显示一个带有指定消息、文本输入字段和确认按钮的对话框。用户可以输入文本，然后点击确认按钮。 |


这些是一些常用的 BOM 属性、对象和方法，它们提供了与浏览器窗口、文档、导航、定时器和用户交互等相关的功能。
