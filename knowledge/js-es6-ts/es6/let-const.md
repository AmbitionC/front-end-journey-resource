| 特性 | let/const | var |
| :---: | --- | --- |
| 块级作用域 | 声明的变量具有块级作用域，仅在声明的块内有效。 | 声明的变量在函数体内有效，不存在块级作用域。 |
| 变量提升 | 不会进行变量提升。 | 变量会进行提升，可以在声明之前访问变量。 |
| 重复声明 | 不允许在同一作用域内重复声明同一个变量。 | 允许在同一作用域内重复声明同一个变量。 |
| 全局对象属性 | 不会将声明的变量添加到全局对象上。 | 将声明的变量添加到全局对象上。 |
| 循环中的行为 | 每次迭代循环都会创建一个新的变量绑定。 | 循环变量会被提升到函数体顶部，存在闭包问题。 |
| 适用场景 | 推荐在任何需要声明变量的地方使用。 | 在 ES6 之前的老版本浏览器兼容性考虑或特殊需求时使用。 |
