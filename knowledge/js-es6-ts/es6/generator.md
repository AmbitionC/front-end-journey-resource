Generator（生成器）是JavaScript中的一种特殊的函数，它允许你编写一个函数，该函数可以多次暂停和恢复执行。这种函数通常用于创建迭代器或处理异步任务的序列。

#### 基本概念
+ **生成器函数**：使用`function*`语法声明的函数，它可以包含多个`yield`表达式。
+ **yield**：`yield`关键字用于暂停函数的执行，并返回一个值。当生成器恢复执行时，它从上次暂停的地方继续执行。
+ **迭代器**：生成器函数返回一个迭代器对象，该对象有一个`next()`方法，用于控制生成器函数的暂停和恢复。



#### 使用场景
1. **创建自定义迭代器**：通过生成器，你可以创建自定义的迭代器来遍历数据集合，如数组、对象或任何自定义的数据结构。
2. **异步任务管理**：生成器可以与`Promise`和`async/await`结合使用，创建更复杂的异步流程控制，如请求多个API并按顺序处理结果。
3. **惰性计算**：生成器可以用于惰性计算序列，即只在需要时才计算下一个值，这有助于节省计算资源和内存。



#### 示例
以下是一个简单的生成器函数示例，它创建了一个迭代器来遍历一个数组：

```javascript
function* iterateArray(array) {
  for (const item of array) {
    yield item;
  }
}

const array = [1, 2, 3, 4, 5];
const iterator = iterateArray(array);

console.log(iterator.next().value); // 输出: 1
console.log(iterator.next().value); // 输出: 2
// ...以此类推
```



#### 注意事项
+ 生成器函数不能直接调用，它们必须通过调用返回的迭代器的`next()`方法来间接执行。
+ 在生成器函数内部，每次遇到`yield`关键字时，函数会暂停执行，并返回`yield`后面的值。
+ 生成器函数可以通过`yield`返回多个值。
+ 当没有更多的`yield`表达式时，生成器会结束执行，并返回一个`done`属性为`true`的对象。
