| 分类 | 描述 | 示例元素 |
| --- | --- | --- |
| 块级元素 | 在页面中显示为独立的块，通常从新的一行开始，充满父元素的宽度 | <div>, <p>, <header>, <footer>, <nav>, <section>, <article>等 |
| 内联元素 | 在行内显示，不会导致新的换行，根据内容大小自动调整宽度 | <span>, <a>, <strong>, <em>, <img>, <input>等 |
| 内联块元素 | 在同一行内显示，可以设置宽度、高度等块级元素的属性 | <button>, <input>, <label>等 |
| 可替换元素 | 显示内容为外部资源，而不是HTML中定义的内容 | <img>, <input>, <select>等 |
| 表单元素 | 用于创建表单，允许用户输入数据并提交到服务器 | <input>, <textarea>, <select>等 |
