#### 媒体查询（Media Queries）： 
+ 使用CSS3中的媒体查询，根据设备的屏幕宽度、高度、方向等特性，为不同的屏幕尺寸应用不同的样式。
+ 例如，可以针对小屏幕设备（如手机和平板电脑）和大屏幕设备（如台式电脑和笔记本电脑）应用不同的布局和样式。

#### 弹性布局（Flexible Layouts）： 
+ 使用相对单位（如百分比、em、rem等）而不是固定单位（如像素）来定义尺寸和间距，使布局能够根据屏幕尺寸自动调整。
+ 例如，使用百分比定义容器和项目的宽度，使它们随着屏幕尺寸的变化而自动调整大小。

#### 网格布局（Grid Layout）： 
+ 使用CSS网格布局（CSS Grid Layout）来创建灵活的多列布局，可以自由调整项目的大小和位置，适应不同的屏幕尺寸和布局需求。
+ 例如，使用grid-template-columns和grid-template-rows属性定义网格的列和行，使用grid-column和grid-row属性定义项目的位置。

#### 弹性图片（Flexible Images）： 
+ 使用max-width: 100%样式将图片的最大宽度设置为容器的宽度，使图片能够根据屏幕尺寸自动缩放，避免图片溢出容器。
+ 例如，设置img标签的样式为max-width: 100%。

#### 媒体资源替换（Media Resource Replacement）： 
+ 使用CSS3中的媒体资源替换技术（如响应式图像、响应式视频等）来根据设备特性加载不同的媒体资源，提高页面加载速度和用户体验。
+ 例如，使用<picture>标签和<video>标签的媒体查询属性来定义不同屏幕尺寸下加载的图片和视频资源。
