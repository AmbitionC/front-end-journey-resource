下表是CSS中常见的定位方式及其区别：

| 定位方式 | 描述 | 特点 |
| :---: | --- | --- |
| 相对定位<br/>Relative | 相对于元素在文档流中的原始位置进行定位，不影响其他元素的位置 | + 元素保留在文档流中，不会对其他元素产生影响。<br/>+ 可以通过top、right、bottom、left属性调整元素的位置。 |
| 绝对定位<br/>Absolute | 相对于最近的已定位的父元素进行定位，如果没有已定位的父元素，则相对于文档进行定位 | + 元素脱离文档流，不占据原有位置，可以自由移动。<br/>+ 通常用于实现菜单、弹出框等需要脱离文档流的元素。 |
| 固定定位<br/>Fixed | 相对于浏览器窗口进行定位，即使页面滚动也不会改变位置 | + 元素脱离文档流，始终保持在相对于浏览器窗口的位置。<br/>+ 通常用于实现导航栏、悬浮广告等始终显示在页面某个位置的元素。 |
| 粘性定位<br/>Sticky | 结合了相对定位和固定定位的特性，当元素到达某个位置时固定定位，否则相对定位 | + 元素在滚动到指定位置时固定定位，否则表现为相对定位。<br/>+ 适用于需要在某个位置固定，但在其他位置表现为普通定位的元素。 |
