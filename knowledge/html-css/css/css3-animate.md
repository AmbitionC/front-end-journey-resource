#### CSS特性
| CSS3特性 | 描述 |
| :---: | --- |
| 圆角（border-radius） | 设置元素的圆角边框 |
| 阴影（box-shadow） | 添加元素的阴影效果 |
| 渐变（linear-gradient、radial-gradient） | 创建元素的渐变背景 |
| 过渡（transition） | 平滑过渡样式的元素状态变化 |
| 变换（transform） | 元素上应用2D或3D的变换效果，如平移、旋转、缩放、倾斜等 |
| 动画（@keyframes） | 定义元素的动画效果，实现复杂的动态效果 |
| 媒体查询（@media） | 根据不同的设备和屏幕尺寸应用不同的样式 |
| 多列布局（column） | 创建多列布局的样式 |
| 字体图标（@font-face） | 引入自定义字体，实现矢量图标的显示 |
| flexbox布局 | 创建灵活的、自适应的布局，简化元素的排列和对齐 |




#### 动画
| CSS3动画 | 描述 |
| :---: | --- |
| 过渡动画 | 实现元素状态变化时的平滑过渡效果 |
| 关键帧动画 | 定义元素的动画序列，实现更复杂的动态效果 |
| 缩放动画 | 实现元素的缩放动画效果，使元素在不改变布局的情况下缩放 |
| 旋转动画 | 实现元素的旋转动画效果，使元素沿着指定轴线进行旋转 |
| 淡入淡出动画 | 实现元素的透明度渐变效果，实现元素的淡入淡出效果 |
