| 属性值 | 描述 |
| :---: | --- |
| block | 元素将以块级元素的方式显示，即元素将会显示为一个块，独占一行，并且会在前后添加换行符。 |
| inline | 元素将以内联元素的方式显示，即元素将会显示为一行，与其他内联元素并排，不会强制换行。 |
| inline-block | 元素将以内联块元素的方式显示，即元素将显示为一行，与其他内联元素并排，但可以设置宽度和高度。 |
| none | 元素将不会被显示，即元素不会在页面上显示，也不会占据任何空间。 |
| table | 元素将会显示为一个表格元素，即元素会以表格的形式显示。 |
| table-cell | 元素将会显示为一个表格单元格，即元素会以表格单元格的形式显示。 |
| flex | 元素将会显示为一个弹性盒子，即元素会以弹性盒子的形式显示，可以方便地进行布局和排列。 |
| grid | 元素将会显示为一个网格容器，即元素会以网格布局的形式显示，可以方便地进行复杂的布局。 |
