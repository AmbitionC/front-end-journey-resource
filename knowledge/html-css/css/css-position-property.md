CSS中的定位属性用于控制元素在页面中的位置，常见的定位属性包括：

+ **position**：用于指定元素的定位方式，常见取值包括： 
    - `static`：默认定位方式，元素遵循正常文档流的布局规则。
    - `relative`：相对定位，元素相对于其自身在文档流中的位置进行定位，可以通过top、right、bottom、left属性调整位置。
    - `absolute`：绝对定位，元素相对于其最近的已定位的父元素进行定位，如果没有已定位的父元素，则相对于文档进行定位。
    - `fixed`：固定定位，元素相对于浏览器窗口进行定位，即使页面滚动也不会改变位置。
    - `sticky`：粘性定位，元素在滚动到指定位置时固定定位，否则表现为相对定位。
+ **top、right、bottom、left**：用于调整元素的偏移量，常用于相对定位和绝对定位的元素，以确定元素在页面中的具体位置。 
+ **z-index**：用于控制元素在堆叠顺序中的层级关系，取值为整数，值越大越靠前，常用于处理重叠元素的显示顺序。
