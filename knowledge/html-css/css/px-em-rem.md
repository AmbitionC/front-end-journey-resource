| 单位 | 描述 |
| :---: | --- |
| px | 像素（Pixel）是相对长度单位，通常被用于网页上的长度值，是相对于显示器屏幕的分辨率而言的，是最基本的长度单位。 |
| em | em是相对单位，相对于其父元素的字体大小。如果没有明确设置字体大小，那么em的大小会继承父元素的字体大小。 |
| rem | rem也是相对单位，相对于根元素（即HTML元素）的字体大小。这意味着它不受父元素的字体大小影响，具有更强的可控性。 |




+ **px**：绝对单位，具体的像素值，不受其他因素影响，适用于边框、间距等固定尺寸的元素。
+ **em**：相对单位，相对于父元素的字体大小，适用于需要根据父元素字体大小变化而变化的元素，如文本。
+ **rem**：相对单位，相对于根元素的字体大小，适用于需要根据根元素字体大小变化而变化的元素，如布局的宽度、高度等。
