Canvas 和 SVG 是用于在网页上绘制图形的两种主要技术，它们有以下区别：

| 特点 | Canvas | SVG |
| :---: | --- | --- |
| 绘图方式 | 基于像素的位图绘图 | 基于矢量的矢量绘图 |
| 绘制方法 | 使用 JavaScript 绘制 | 使用 XML 标记描述图形 |
| 图形元素 | 位图图形（像素） | 矢量图形（路径、线段、圆、矩形等） |
| 渲染性能 | 适用于复杂的、动态的图形 | 适用于简单的、静态的图形 |
| 缩放和变换 | 相对复杂，可能会导致图像失真 | 相对简单，不会导致图像失真 |
| 事件处理 | 需要自行实现事件处理逻辑 | 直接支持事件绑定和处理 |
| 文本支持 | 有限，支持基本的文本绘制 | 完整支持文本绘制，可进行样式控制 |
| 网页访问 | 将绘制结果保存为图片文件或数据URL | 直接嵌入在 HTML 中，可通过 CSS 控制 |
| 编辑和交互性 | 编辑较为困难，交互性较低 | 编辑方便，交互性强 |
