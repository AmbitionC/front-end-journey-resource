`useCallback` 和 `useMemo` 都是 React Hooks 提供的性能优化工具，用于避免不必要的重新计算和重新创建，从而提高组件的性能。

+  `useCallback` 用于优化函数的性能，通过缓存回调函数并在依赖项不变时返回缓存的函数引用。常见的用例包括将回调函数传递给子组件时，避免子组件的不必要重渲染，以及将回调函数作为依赖项传递给 `useEffect` 或 `useMemo`，避免因为重新创建而导致的不必要的触发。 
+  `useMemo` 用于优化计算的性能，通过缓存计算结果并在依赖项不变时返回缓存的结果。常见的用例包括将计算结果作为组件的 prop，避免不必要的重新渲染，以及缓存计算代价较高的结果，避免重复计算，提高性能。 

在使用 `useCallback` 和 `useMemo` 时，需要注意传递给它们的依赖项数组，以确保在依赖项发生变化时触发相应的更新。同时也要注意避免过度使用，只在性能瓶颈明显的情况下才考虑使用这两个优化方法，以保持代码的简洁和可读性。
