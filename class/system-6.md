## 一、数据可视化领域

### 经典产品

1. **阿里 AntV 系列**（G2/G6/F2）

   * **团队**：蚂蚁金服前端视觉化团队
   * **特点**：

     * G2 用于统计图表，G6 用于图（Graph）可视化，F2 针对移动端
     * 基于 **可声明式渲染**，支持自定义图形和动画
   * **底层知识**：

     * **渲染引擎**：Canvas / SVG / WebGL
     * **布局算法**：

       * 力导向布局（力学模拟节点和边的排布）
       * 树形/层级布局（如树状图）
     * **性能优化**：

       * Canvas 批量绘制
       * 图表数据虚拟化

2. **ECharts**

   * **团队**：百度前端团队
   * **特点**：

     * 丰富的图表类型，支持移动端和大屏
     * 内置交互能力（缩放、拖拽、数据高亮）
   * **底层知识**：

     * 数据驱动渲染
     * Canvas 渲染管线
     * 数据预处理算法（如分层聚合、抽样优化）

### 典型应用

* 实时监控大屏：物流调度、金融风控
* 交互式分析：BI 可视化


## 二、音视频和实时通信领域（RTC）

### 经典产品

1. **Zoom / 腾讯会议 / DingTalk 视频会议**

   * **团队**：企业前端与底层音视频研发团队
   * **特点**：

     * 高并发视频流传输
     * 多端一致体验
   * **底层知识**：

     * **WebRTC 协议**：P2P 连接、ICE/DTLS/SRTP
     * **编码器/解码器**：H.264/H.265、Opus
     * **流控算法**：自适应码率（ABR）、延迟优化
     * **多路混流**：音视频合并和转发

2. **Agora（声网）**

   * **团队**：RTC 云服务团队
   * **特点**：

     * 提供 Web SDK / Native SDK / 云端 RTM
     * 实时互动低延迟
   * **底层知识**：

     * 网络抖动补偿
     * UDP 数据传输、FEC 前向纠错
     * 音视频同步机制


## 三、3D 与游戏前端领域

### 经典产品

1. **Three.js**

   * **团队**：开源社区（原作者 Ricardo Cabello）
   * **特点**：

     * WebGL 封装库，降低 3D 渲染门槛
     * 支持材质、光照、物理模拟、动画
   * **底层知识**：

     * GPU 渲染管线（顶点着色器、片段着色器）
     * 空间数据结构（八叉树、BVH）
     * 渲染优化：裁剪、实例化渲染、LOD（Level of Detail）

2. **Babylon.js**

   * **团队**：微软开源团队
   * **特点**：

     * 全功能 WebGL 引擎
     * 支持物理、粒子、PBR（Physically Based Rendering）
   * **底层知识**：

     * 光照计算模型
     * GPU 加速动画
     * 场景管理和渲染队列优化

### 典型应用

* 产品 3D 展示（家居、汽车、电商）
* Web 游戏开发
* VR/AR 应用（WebXR）

## 七、前端垂直领域技术总结

结合业界经典产品和团队经验，可以总结以下几点：

1. **底层算法与数据结构是核心**

   * Diff 算法（Virtual DOM）
   * 优先队列（Fiber调度）
   * 力导向布局（可视化图表）
   * 八叉树/空间分区（3D 渲染）

2. **性能优化必备手段**

   * 虚拟化列表、分块渲染
   * requestAnimationFrame 节流
   * Web Worker 并行处理
   * Canvas/WebGL 批量渲染

3. **工程化和组件化能力**

   * 模块化构建（Webpack/Vite）
   * 动态组件渲染引擎（低代码平台）
   * 状态管理与事件调度

4. **实时与高并发场景**

   * WebSocket / WebRTC / RTC
   * 异步调度与优先级队列
   * 网络优化与流控算法
